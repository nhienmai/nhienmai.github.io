Data Cleaning

In my data cleaning approach, I begin by analyzing the dataset's structure to identify any quality issues such as completeness, validity, accuracy, and consistency, and tidiness issues, which involves optimizing how data is arranged across rows, columns, and tables for maximum clarity.

I employ a "Define-Code-Test" methodology in my cleaning step: "Defining" each problem in text to aid understanding across technical and non-technical stakeholders; "Coding" with precision to implement the necessary fixes; and "Testing" to ensure the data meets the highest standards of cleanliness. This systematic process not only refines the data but also enhances its reliability for insightful analysis.



Data Exploration

My data exploration process begins by defining the key questions based on the business problem. I use SQL to perform detailed data analysis, from basic to advanced queries.

Additionally, I also apply Python libraries to explore the data further and create visualizations for clearer insights.



Data Visualization

In addition to utilizing Excel, Python libraries like Matplotlib and Seaborn for visualization, I also employ Tableau to craft interactive visualizations. This tool enhances comprehensibility through dynamic features like filtering, making the data accessible and engaging for both technical and non-technical audiences.
